Bem-vindo ao Meu Projeto Git

Sobre o Projeto

Este projeto é um exemplo para praticar o uso do Git e GitHub. O arquivo HTML demonstra a estrutura de um site básico com um layout simples e algumas regras de estilo CSS.

Aqui estão algumas das características do projeto:

  • Header com estilo básico
  • Conteúdo principal com padding e layout centralizado
  • Estilos CSS para uma aparência limpa e moderna

O objetivo é praticar a criação de branches, commits e gerenciamento de código-fonte com Git.
